== Description ==

Permite preluarea cursului BNR (Euro, USD, CHF, GBP, XAU) si afisarea lui pe blog.

== Instalare ==

1. Dezarhivati
2. Uploadati fisierul curs_bnr.php in directorul `/wp-content/plugins/`
3. Activati
4. Puneti codul: <?php curs_bnr(); ?> unde vreti sa apara cursul.
6. Si atat.

Daca apar ceva probleme sau vreti sa apara/dispara ceva din curs dati-mi de stire.
